📱 WhatsApp automático

¿Qué es esto? Permite que la app mande el marcador automáticamente a un grupo de WhatsApp mientras el partido está en curso.

  1. Ve a green-api.com y crea una cuenta gratuita.
  2. En el panel, crea una nueva instancia y copia el Instance ID y el API Token.
  3. Escanea el QR que aparece con tu WhatsApp (igual que WhatsApp Web). Solo hay que hacerlo una vez.
  4. Pega aquí el Instance ID y el Token y pulsa Guardar.
  5. En el marcador, pulsa el botón de WhatsApp para elegir el grupo y activar el envío.

💡 El plan gratuito de Green API incluye 3.000 mensajes/mes, más que suficiente para partidos.
